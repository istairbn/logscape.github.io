Manager

The Manager runs the following services

  • Dashboard - The dashboard is the web front-end and is sometimes called the web console. All the search analytics and system administration is done using this component. Connect to the dashboard on port :8080
  • Index Store - The Manager runs an Index Store by default. All Forwarders will send data to the Manager's Index Store.
  • Syslog Server - A Syslog Server runs on the Manager and will accept data from any syslog compatible host.
  • Socket Server - A Socket Server accepts data streamed to a TCP port.

The Failover

The Failover runs exactly the same services as the Manager. On startup, the Failover will copy the managers configuration and also start an Index Store.

and other essential services such as alerting, maintain a copy of the configuration and also run the web console. To continue using the system point your browser to the Failover dashboard ( http://Failover-Address:8080 )

Forwarder

A Forwarder's role is to collect data and stream it off to an Index Store. A forwarder has no capability to participate in searches or index any local data.

Index Store

An Index Store collects data form multiple sources. It also indexes forwarded data and participates in distruted searches.

Indexer

An Indexer watches local data, and participates in distributed searches.
