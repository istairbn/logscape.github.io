Navigating Raw Events (click on a word and update your search )

After a search excecutes, the raw events are displayed in a table below the charts. The log events table provides a
  • Raw View - which displays the log data as it appears. There is a link below each event which will open the file that contributed the line.
  • Data Type View - This shows the log events in tabular structure determined by a datatype mapping
