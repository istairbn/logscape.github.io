Alerts >

Agent Died .

This is an alert which notifies administraters when an agent has expectedly left the environment. This may be happen due to a reboot event, or the host going offline or network problems. A decision needs to be made whether the adminsitrator wants to be notified immiediately when as agent dies or if the notification should be sent after a threshold has been breached.
