How to Deploy an App

Monitor the health of your Infrastructure using Apps

  1. Download an App from the Apps page. Logscape Apps

  2. Go to Configure and find the Deployment section.

  3. Click on the Upload button and select the files that you would like to upload. e.g UnixApp-1.0.zip

  4. Once the file has been downloaded it will appear in the table below. p

    Find the App then click 'DEPLOY'. The status will change fro UNINSTALLED to DEPLOYED.

  5. Each App has a Home Workspace. e.g The UnixApp has the Unix Home workspace.
