Authenticating

Logscape authenticates with your Active Directory or LDAP Server using the settings in this form.

A Logscape System Account will need to be set up for Active Directory with enough credentials to search the LDAP tree.

Users

This section configures which users in your LDAP or Active Directory will be imported into Logscape.

You will need the fully qualified path of your Users.

BaseCN: This is where your LDAP search starts.

UserCN Defines the LDAP folder where your users are located. This is the full qualified path without the BaseCN.

Filter: If the filter is left blank all the users in that path will be pulled in. You can restrict the users list by using a filter. e.g

(memberOf=CN=PowerUsers,OU=Groups,OU=Company,DC=logscape,DC=local) Use the filter to limit users belonging to a particular group or on any other criteria.

Testing

You can test your ldap configuration before saving by clicking the 'Test' button.

Any problems with authentication or search for users will be display in the Output section of the page.

Switching to LDAP

Once you are happy with the configuration change the current active user scheme to 'External LDAP' and then click apply.

This option is used for both Active Directory and LDAP.

Click 'Save' to commit all your changes.
