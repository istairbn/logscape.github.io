Port Considerations


Set up your firewall to allow bi-directional traffic on the following ports. You can modify the base port for the Manager in the boot.properties file. The table below uses the default base port of 11000


Manager

Port Ranges Services
8080Web Dashboard
11000-11100Host Services
1468,1516 Logscape Syslog Server. udp,tcp
15000,25000 LookupService

Agent

11000-11100 IndexStore,Indexer,Forwarder
