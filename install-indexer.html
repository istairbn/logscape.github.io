Install an Indexer


The Indexer will index and monitor files locally and participate in searches. Use an Indexer to speed up searches of large data volumes or where a Forwarder is not practical

Note: Do not use OpenJDK. There are known performance problems with this JVM and Logscape may not run as expected.

Duration: Just under 5 minutes

Prerequisites: Make sure that you have Oracle Java 7 and above and that Java is accessible from your path

  1. Download Logscape-2.0.zip and unzip

  2. Navigate to your Logscape installation and inside the scripts folder, type:

  3. A prompt will appear to configure your installation. Choose option 2) for a Indexer

  4. Next you will be asked for the address/hostname of your Manager. A quick test is executed to check the connectivity between your Indexer and the Manager.

  5. Once the test is passed you can accept the default configuration for the other parameters by pressing enter.

  6. After you have finished configuring navigate back to the Logscape home directory and type the command below to start the Manager

    logscape $> ./logscape.sh start

  7. Login as sysadmin or admin and check if the Indexer appears in the agent list on the Agents page
