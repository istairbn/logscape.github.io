Data Groups

A Data Group defines a user's access to data. A Data Group contains a list of data sources to include and exclude.

The list of what to include can contain a valid data source or a data group.

tag:DATASOURCE

or

group:DATAGROUP

using this syntax nested groups are possible allowing for a complex hierachical mapping of data access roles for users
