Navigating Raw Events

When a search executes, the raw events are displayd in a table below the charts. The log events table provides a
  • a Raw View - Which displays the log data as it appears. There is a link below each event which will open the file that contributed the line
  • a Data Type View - This shows the log events in tabular structure determined by a datatype.
