The home page

This is your default workspace. Customize it to show information metrics and charts of information you are interested in.

Use the menus on the left to Navigate

From here you can navigate to different parts of Logscape

  • Home - will take you to the Home workspace
  • Workspace - Allows you to create your own workspace and use Logscape Apps
  • Search - Visualize your data
  • Configure - Create data sourcesi, configure alerts and administer other components of the system

Navigating Raw Events (click on a word and update your search )

After a search excecutes, the raw events are displayed in a table below the charts. The log events table provides a
  • Raw View - which displays the log data as it appears. There is a link below each event which will open the file that contributed the line.
  • Data Type View - This shows the log events in tabular structure determined by a datatype mapping
