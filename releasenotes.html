Release Notes

Release 2.0.4(06-December-2013)

  • Stability improvement have been made by moving away from Krati(LinkedIn) to Persisit. All thread interrupts are prevented they can potentially lead to journal corruption
  • HTML enhancements with GeoIP and D3 DataMaps
  • Table - Cell heatmap support to allow customization
  • WindowsApp and UnixApp refinements to reduce data volumes as improve accuracy.

Release 2.0.3(22-November-2013)

  • Substantial performance improvements (and related blog post)
  • Wrestling with KV Store implementation and reindexing
  • Improved HTML5 Workspace switching performance

Release 2.0.2(23-October-2013)

  • Minor enhancements to improve the initial User Experience

Release 2.0.1(18-October-2013)

  • General performance improvements on search throughput
  • IndexStore & Managers will now consume native Memory from the Host in order to maximise performance and Indexing capability
  • Chart Overlay support has been added (overlay bar & line charts etc)
  • Latest version of Rickshaw.js being used for common chart types
  • Better browser performance where workspaces were consuming background resources
  • Workspace minimum height is now being applied and maintained
  • Improved eventing and thread/port consumption from Indexer Stream Reduce to AggregationEngine instances
  • Field.txn(elapsed) now breaks down elapsed durations into H,M,S

RC4 (14-August-2013)

  • Improved pan and zoom controls on the Search Page
  • Improved layout on the Agents page
  • Removed doublespacing bug from SysLog Event Handler
  • Fix buttons on workspaces, search, configure, to work in more browsers
  • Create/Edit/Delete Resource Groups on the Agent tab
  • Improved Dynamic Field discovery to work with XML tags
  • Fixed ActiveDirectory integration
  • - EULA Enforcement
  • - Pie-chart sizing problem fixed
  • Added workspace pie-legend toggle
  • Fixed countUnique problems in chart(table)
  • Tightened control around logscape/pids directory killing

RC3 (02-August-2013)

  • Support workspace browser-resize
  • - Move workspace controller to the workspace header
  • Improved Dynamic Field performance (30%)
  • Fixed problem with buckets(1) wasnt rolling correctly on long searches
  • 'chart(table)' layout fix on workspace widget, added select support
  • Make file-'tail' only allow access to tailing once the full browser payload is downloaded and page-ready has fired
  • Newer version of MapDB (due to bug on unsupported operation affecting event handling on WebTier)
  • Improvements to date-time locale preferences and parsing/extraction
  • Improved search performance on large files and kv execution
  • added chart(line-connect) to join line points between missing items
  • Linking between search-legend and events table
  • Much improved disk-io performance

RC2

  • Fixed many ui click handlers
  • Improved overall stability
  • Added wildcard support on DataSources
  • Improved DataSource browser functionality
  • Printing support
  • Https support added
  • added Proxy/via Manager to stream remote data from any connected agent
  • disable file deletion on IndexStore when deletion detected at source
