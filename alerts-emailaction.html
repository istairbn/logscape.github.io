Configuring Email

If the Email Alert action is configured and email notification is sent to stackholders when the alert is triggered. A type email has the following format

Alert Details - It will include the name of the Search and the number of trigger events that fired the alert

see attached Report[Coherence - Grid Network Low Node Tx/Rx] Trigger[3] ===================================================

Raw Alert Event - The last message to trigger the event as it appears in your log data is the next part of the email.

Last Msg:/var/dg-logs/coherence/lt/session-caching/report/2012072400-network- health.txt,33,Event: fields[[coh-net, WOK-ENV-COH03, 2012072400-network-health.txt, /var/dg-logs/coherence/lt/session-caching/report/2012072400-network-heal th.txt, lt-sc, CohJmxLTAgent] 269044 Tue Jul 24 00:31:21 BST 2012 0.8937093275488069 1.0 0.919175911251981 1.0]

Trigger Events - The trigger events are the final part of the email. The trigger events follow the datatype fields assigned to the log data. The system fields are included at the end.
