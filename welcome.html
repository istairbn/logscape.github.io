Manager Installation

The Manager monitors and visualizes your data and also runs a Syslog Server (udp:1468;tcp:1516)

On Unix

Download Logscape-{VERSION}.zip from the download page and then run "logscape.sh start".

On Windows

Download the MSI and choose the Manager Role.

Forwarder Installation

You can now install Forwarders on several hosts to collect data from your applications and devices. The data will be streamed to the Manager.

On Unix

Download the Logscape-{VERSION}.zip to a new host. Unzip and then navigate to the $LOGSCAPE_HOME/scripts folder to configure the Logscape Agent. Run the configure.sh and then choose the "Forwarder" role. Once the setup has completed start the Forwarder by typing: "logscape.sh start"

On Windows - Logscape.MSI:

Copy the MSI to a host, install and then choose the "Forwarder" Role. The installer will automatically run the Forwarder as a service.

First Time Login:

To connect to the manager type the following url in any modern HTML5 browser (e.g Chrome, Safari, IE10, Firefox) http://{managers-host-name}:8080
