Checklist

  • Oracle Java 6 or 7. Use Java6u23 and above.
  • Configure your firewall. Port Considerations
  • Ensure that your servers are clock synced
  • The Manager should not be installed on a virtual machine.
  • (Unix/Linux) Have sar installed, usually found in the systat package
  • Open JDK is not supported. Even though Logscape will start, you may experience performance problems
  • Do not install the 32bit MSI or 64-bit Java
