Workspaces


Related searches are collected onto Workspaces to provide a cohesive view of your data.

Workspace Controls

  • Edit Mode - This button puts the workspace into Edit Mode allowing you to add an delete searches or link other workspaces.
  • Open - Open other workspaces.
  • Print - Exports the current workspace as a PDF document
  • The Navigation Menu - This icon appears in all sections and allows you to navigate to the Configuration Page, to perform system administration tasks or you can navigate to the Workspaces, Search and Home pages.
