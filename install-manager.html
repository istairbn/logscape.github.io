Installing the Manager

Duration: Just under 5 minutes Prerequisites: Make sure that you have Oracle Java 7 and above and that Java is accessible from your path


Note: Do not use OpenJDK. There are known performance problems with this JVM and Logscape may not run as expected.

  1. Download Logscape-2.0.zip and unzip
  2. Copy your license to your Logscape location.
  3. Inside the Logscape parent folder type logscape $> ./logscape.sh start
  4. It will take about 30 seconds for the manager to fire up all the services. Point your browser to:http://your-logscape-host:8080
  5. Once the Login appears. Use the following credentials for first time access to your systemUser: admin Password: ll4bsThe sysadmin account gives you access to all data on the system, including Logscape specific log data. The admin account gives you access to all data on the system but omits any Logscape specific data in your search results.
